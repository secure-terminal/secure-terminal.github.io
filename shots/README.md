# Screenshots on this site - how they are generated (do not hand-edit)

These PNGs are produced by committed generators. To update a shot, RE-RUN the
generator; do not paint or hand-crop.

## This directory (`shots/`) - the paste/copy review bar

`paste-warning.png`, `copy-warning.png` render the real
`secure_terminal.review.ReviewBar` headless (offscreen Qt `grab()`, deterministic).

Regenerate both with one command (from dist-ai):

    secure-terminal-shots [OUTPUT_SHOTS_DIR]     # defaults to this dir

which drives `usr/share/secure-terminal-tests/paste-warning-shot.py <out.png>
[paste|copy]`. Add a line to the script's spec loop to add a shot.

## `comparison/shots/` - the terminal-comparison shots

Real Debian terminal emulators (and secure-terminal) under a nested labwc
compositor. Generated by `comparison/capture.sh` in the terminal-poc-corpus repo
(see its `comparison/README.md`); needs an X server + labwc, so regenerate it in
the sandbox with `ST_REPO=<secure-terminal checkout>`.
